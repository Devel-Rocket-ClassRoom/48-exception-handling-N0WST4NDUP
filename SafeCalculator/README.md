# 안전한 나눗셈 계산기

안전한 나눗셈 계산기를 구현하세요.

사용자가 두 정수를 입력하면 나눗셈 결과를 출력합니다. 단, 다음 상황에서 예외를 처리해야 합니다:

1. 숫자가 아닌 값을 입력한 경우 → "올바른 숫자를 입력하세요." 출력
2. 0으로 나누려고 한 경우 → "0으로 나눌 수 없습니다." 출력
3. 정상적인 경우 → 나눗셈 결과 출력

모든 경우에 "계산기를 종료합니다."가 출력되어야 합니다 (finally 사용).

### 요구사항

- `SafeCalculator` 클래스에 `Divide(string num1, string num2)` 메서드 구현
- 문자열을 정수로 변환 후 나눗셈 수행
- FormatException과 DivideByZeroException을 각각 처리
- finally 블록에서 종료 메시지 출력

## 예상 실행 결과

**테스트 1: 정상 입력**

```
=== 테스트 1: 정상 입력 ===
10 / 2 = 5
계산기를 종료합니다.
```

**테스트 2: 0으로 나누기**

```
=== 테스트 2: 0으로 나누기 ===
0으로 나눌 수 없습니다.
계산기를 종료합니다.
```

**테스트 3: 잘못된 형식**

```
=== 테스트 3: 잘못된 형식 ===
올바른 숫자를 입력하세요.
계산기를 종료합니다.
```
